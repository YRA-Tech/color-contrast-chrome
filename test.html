This page contains various contrast scenarios for testing the extension.

Test Instructions

  1. Open the extension popup
  2. Try "Capture Visible Tab" in both Hardware and CSS pixel modes
  3. Try "Capture Whole Page" in both modes
  4. Try "Capture Selected Area" on different sections
  5. Verify the rescan function works with different settings
  6. Check that low contrast areas are properly highlighted
